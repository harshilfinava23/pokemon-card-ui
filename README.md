# Pokemon Card UI

A Pokemon Gen I trading card UI built with pure **HTML & CSS**.  
This project focuses on understanding **CSS Position Properties** in real UI layouts.

🔗 **Live Demo:** [pokemon-card-ui.vercel.app](https://pokemon-card-ui.vercel.app)

## About

The goal was to recreate a given UI design as close as possible using:

- `position: relative`
- `position: absolute`

No JavaScript. No frameworks. Pure HTML & CSS only.
